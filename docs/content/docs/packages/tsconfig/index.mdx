---
title: TSConfig
description: TypeScript Configuration for Moeru AI.
---

```package-install
npm i -D @moeru/tsconfig
```

```jsonc
// tsconfig.json
{
  "files": [],
  "references": [
    { "path": "./tsconfig.app.json" },
    { "path": "./tsconfig.node.json" }
  ]
}
```

```jsonc
// tsconfig.app.json
{
  "extends": "@moeru/tsconfig",
  "include": ["src"]
}
```

```jsonc
// tsconfig.node.json
{
  // or "@moeru/tsconfig/tsconfig.node.json"
  "extends": "@moeru/tsconfig/node.json",
  "include": ["*.config.ts", "scripts"]
}
```
