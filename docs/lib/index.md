# @moeru/std

Standard TypeScript Library for Moeru AI.

```bash
pnpm add @moeru/std
```

## Libraries suitable for use with `@moeru/std`

Some utility functions may not be included in `@moeru/std`, we recommend the following libraries as a supplement:

- [SukkaW/foxts](https://github.com/SukkaW/foxts)
- [unjs/std-env](https://github.com/unjs/std-env)
- [unjs/scule](https://github.com/unjs/scule)
